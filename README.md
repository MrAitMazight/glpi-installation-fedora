# GLPI 11 Installation on Fedora 43

This project explains how to install GLPI 11 on Fedora Linux using Apache, MariaDB and PHP.

## Environment

- Fedora 43
- Apache Web Server
- MariaDB
- PHP
- GLPI 11

## Architecture

Fedora 43
│
├── Apache (Web Server)
├── PHP
├── MariaDB (Database)
└── GLPI 11

## Installation Steps

1. Install Apache
2. Install MariaDB
3. Install PHP
4. Create Database
5. Install GLPI
6. Configure permissions
7. Access GLPI via browser
